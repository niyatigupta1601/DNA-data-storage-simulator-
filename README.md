# 🧬 DNA Data Storage Simulator

## About

The DNA Data Storage Simulator is an educational web application developed as a college project to demonstrate the concept of **DNA-based data storage** through **biomimicry**. The application shows how digital text can be converted into binary and then encoded into DNA sequences, illustrating how biological molecules can inspire next-generation data storage technologies.

The project aims to simplify a complex scientific concept through an interactive and easy-to-understand interface.

---

## Features

- 📝 Convert user-entered text into binary representation
- 🧬 Encode binary data into DNA nucleotide sequences (A, T, C, G)
- 💡 Demonstrate the concept of biomimicry in technology
- 📚 Educational visualization of DNA data storage
- 🎨 Interactive and user-friendly interface

---

## Problem Statement

Traditional storage devices have limited lifespan and storage capacity. DNA has emerged as a promising medium for long-term, high-density data storage due to its stability and enormous information capacity.

This project provides a simplified simulation to help students understand how digital information can be represented and stored using DNA sequences.
